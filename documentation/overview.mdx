# Overview

Welcome to the College DAO documentation. This section provides comprehensive guides for joining our community, participating in governance, and contributing to the ecosystem.

## Documentation Structure

### [Get Started](./get-started)
Your complete onboarding guide to College DAO. This guide covers:
- Setting up a Solana wallet
- Creating your club profile
- Getting sponsored and joining the council
- Submitting your first proposal
- Contributing back to the DAO

### [Governance](./governance)
Learn about College DAO's council model, voting mechanisms, and how to participate in the decision-making process. This guide covers:
- Council structure and membership
- Proposal creation and voting
- GitHub and on-chain governance integration

### [Treasury](./treasury)
Understand how College DAO manages funds across multiple chains. This guide includes:
- Multi-chain wallet addresses
- Donation guidelines
- Fund allocation principles
- Transparency requirements

## Quick Navigation

- **New club joining?** Start with [Get Started](./get-started)
- **Want to understand governance?** Read [Governance](./governance)
- **Ready to contribute?** Check the [Treasury](./treasury) guide
